[閱讀] 任何人都適用的完美學習法

融合《快思慢想》《刻意練習》等多本著作精華的完美學習法

2021/03/15

J-Ray C

從小不停的考試，你真的掌握了學習的方法嗎？真的有任何人都適用的完美學習法嗎？如果有，那又該如何掌握呢？

這本書在說什麼？

不像一般學習書強調快速記憶法、時間管理或考試技巧，而是從「動機」與「思維模式」開始，結合心理學與教育研究釐清學習背後的驅動力，建構出屬於自己、長久可用的學習方式。

真正有效的學習，是建立在「理解自己如何學習」的基礎上。

從動機設定、情緒管理、長期目標、心智模型、自我效能、休息與創意、讀書方法等層面，全面且務實地提供操作建議，幫助我們更有效率、更有樂趣地學習，並把學習轉化為一種內在力量。

我從這本書獲得什麼？

找回對學習與未來的期待感

對於自己正在做的事情賦予意義，確信其價值，才能擁有撐下去的力量。

有時會陷入「做這件事到底有什麼意義？」的狀態，此時主動「賦予意義」給自己一個繼續走下去的理由。

搭配「小成就會累積大成果」的思維，能更有勇氣以微小但穩定的步伐前進。

建立「後設認知」與「自我效能」的意識

這兩個術語原本對我來說有點抽象，但書中的解釋非常具體：

  • 自我效能：相信自己有能力組織完成某個目標的行動方案
  • 後設認知：客觀看待自己的能力，清楚「我知道什麼，不知道什麼，我的行動會引起什麼後果」

動機優先，方法其次

動機才是問題的解答，不是方法。

當卡住學不下去時，第一步不是去找解法，而是回頭看自己：
我為什麼想做這件事？我真正想解決的是什麼？這個目標對我有什麼意義？

長期目標的拆解術

長期目標不是幻想，而是今天不斷實踐的累積。

與其立一個大又遠的目標，不如把它拆解成可以實際描述的中短期目標，最好是今天可以做的那種。

休息是解決問題之鑰

潛意識思考很高機率能擺脫原有線性思維邏輯，創造創意思維邏輯。
面對解不開的問題或要做出棘手決策的時刻，還有靈感枯竭時需要充足休息才能使潛意識得以發揮力量。

深度學習與創意來自挑戰與失敗

  • 越困難的學習越難忘
  • 多挑戰多失敗，更能刺激創造力

自認沒創意的人，比起增加新點子的數量，更執著於反覆改善舊點子。
真正的創意人是會先想出許多點子。

優秀心智模式的三個條件

  1. 專注力
  2. 接受回饋
  3. 有意識地修正

成為創意人士的四個開始

  1. 培養和自己專業領域無關的興趣
  2. 去海外旅遊
  3. 多件不同的人
  4. 閱讀各種書籍

多種有效的讀書法

  • 慢讀：全面剖析一本書，歸納重點、閱讀作者其他著作、寫下自己的想法
  • 觀讀：抱持觀點閱讀其他書籍，在不同中類的書中得到共同觀點的啟發
  • 再讀：經過一定的時間，遺忘後再重讀
  • 朗讀：朗讀是文章的最後投手

閱讀後進行了什麼行動？

建立「自我教練」的感覺

在每一次學習時多問自己幾個問題：

  • 我現在真的理解了嗎？
  • 我為什麼這樣做？
  • 我接下來應該怎麼調整？

培養讀書習慣的七種方法

  1. 遠離手機：離手機越遠，則離閱讀越近
  2. 特定場所：固定場所能有效培養閱讀習慣
  3. 利用認知失調：我在閱讀是因為我喜歡
  4. 一次買大量的書：堆滿了沒讀過的書的書房才有誘惑人讀書的魅力
  5. 讀書會：多和愛書人士相聚，有助於提升生活品質
  6. 一次讀三到四本書：書很無趣的時候就換一本想看的轉換心情
  7. 待在愛看書的身旁：刺激閱讀慾望、得到好書資訊

這本書適合什麼樣的人閱讀？

  • 對「學習」感到焦慮或厭倦的人
  • 不知道該怎麼設定目標、總是懷疑自己方向對不對的人
  • 想學習新領域，卻不知從何開始的人
