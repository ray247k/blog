[閱讀紀錄] 下班後1小時的極速學習攻略

職場進修達人不辭職，靠「偷時間」高效學語言、修課程，10年考取10張證照

2022/06/28

J-Ray C

不辭職，光用下班零碎時間進修、精通技藝
利用碎片時間，達到最大成效

目標設定

任何事都得試過才知道
珍惜小確幸，才能繼續成長

有效設定目標，讓成功機率提升

做任何一件事必然都有其原因，而目標中「有多少自己的想法」決定了努力的程度
如果目標是為了自己，勢必會更加努力

任何的選擇都與失敗無關

如果已經盡了最大的努力設定明確的目標，並且經過深思熟慮後做出最適當的決定
那個這個決定已經無關失敗，「不是決定錯誤，只是和目標無緣」。不必再對此感到挫折

利用有限的機會盡快做出決定

盡快掌握一個機會，那個下一個過會就在不遠處了
一旦做出決定就盡所能努力，千萬別再回頭

意志力

面對二度學習，四點強化意志力

  1. 尋找身邊有實力的前輩或同事
  2. 階段性啟動學習
  3. 從有興趣的領域下手，提高學習的意志和興趣
  4. 停止生活中無意義的行為

職場人學習失敗的四大原因

  1. 先後順序設定失敗：必須把學習放在社交或是休閒活動前
  2. 挑戰爬不上去的樹：準備有機會通過的考試
  3. 完美主義作祟：及格就好！當職場人的學習出現壓力的那一刻起，學習就註定失敗
  4. 喪失學習的意志力：重新檢討自己想做的事情和目標，找回學習的欲望

學習法

比起反覆閱讀、默背，更多證據顯示反覆提取的效果更好

戰勝一開始對學習的抗拒

在克服抗拒心理的同時，看見自己的努力開花結果

  1. 利用一開始決定認真學習的決心，盡最大努力快速學習及大量學習
  2. 努力學習到底
  3. 為了建構知識體系，必須熟悉各種專有名詞

面對考試的態度

適時檢視自己的實力有無因學習而提高

  1. 謙虛、沈著的態度面對考試
  2. 精準且具體掌握通過考試不可或缺的要素
  3. 養成邊思考邊學習的習慣
  4. 逐步縮減學習的範圍
  5. 拒絕不良的生活習慣

一個人個成就有多大，全繫於他下了多大的決心
堅定的決心背後，必須有明確的目標和具體的計劃做為後盾

開始學習階段

常勝軍特徵

  1. 學習目標清楚明確
  2. 持續檢視個人實力有無因學習而提高
  3. 知道自己一天的學習量有多少
  4. 不養成破壞學習的不良生活習慣

常敗軍特徵

  1. 學習目標不明確，經常三心二意
  2. 花更多時間在擬定計畫，而非學習本身
  3. 以為坐在書桌前就是學習
  4. 幻想在考場上會有考的結果

正式學習階段

適時檢視自己的實力有無因學習而提高

考前收尾階段

常勝軍特徵

  1. 專注複習各科目中的基本概念
  2. 加強不熟悉的部分，減少背記的分量
  3. 減少需要學習的分量，有助於提高信心

常敗軍特徵

  1. 對科目架構掌握不充分，急著背下所有內容
  2. 因為不安而將重點放在困難且無關緊要的部分
  3. 不斷上網查考試經驗，無法克制不安的情緒

考試結束後

需要一段「緩衝期」重新設定日後計畫

戰勝挫折和不安

進入職場工作後如果還抱著「辛苦學習就能有所回報嗎？」的懷疑心態學習，實力將難以大幅提升
感到挫折時，我們只有兩條路可以選擇。一是直接放棄，二是試著堅持到最後

預防學習失敗的方法

  1. 要有確實的學習動機才能堅持到最後
  2. 既然下定決心學習，就要放下挫折感和不安感
  3. 遭遇心理上的困境時，妥善解決後再學習也不遲

獨自學習時的注意事項

  1. 不和他人比較
  2. 檢視自己的學習是否有助於提升實力
  3. 兼顧學習的質和量

處理情緒

將結果和情緒分開來
情緒波動只會使你一無所有

面對失敗

比起失敗，真正該感到丟臉的是不努力的過程
