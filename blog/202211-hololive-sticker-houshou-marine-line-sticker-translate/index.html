寶鐘瑪琳 Line 貼圖翻譯

船長自己畫的貼圖上架啦！

2022/11/05

J-Ray C

船長在直播上說第一次上架還被 Line 打槍了 7 張，原因大概都是太色
我怎麼就一點都不意外呢

貼圖這邊買：hololive sticker “Houshou Marine”

瑪琳船長前幾天的昭和老人會十分有趣，經歷過 CRT 電視的人應該都被勾起回憶
那個 4:3 的螢幕比例、畫質、字幕、舞步、選曲、主持人說話的風格都十分講究
想起來以前圍在電視前面看超級變變變的時候，而且來賓陣容也很大
