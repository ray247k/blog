[閱讀] 發現我的多重職涯組合

經營最佳職涯與多元身分，一生平衡工作、生活和夢想

2025/01/07

J-Ray C

在不確定的世界中想找到個人與職業平衡的人所寫的指南
點出了未來世代需要面對的職涯發展課題，把人生當成一間公司在規畫，自己就是人生的 CEO！

這本書在說什麼？

因應現代社會中，變化成為新常態，創新的速度不斷加快。而努力工作唯一能保證的是無償加班，而不是成功。作者以他的經驗與觀察引入了「投資組合的人生」這個概念，來回覆「該怎麼維持生活，而且過得幸福快樂？」這個問題。除了說明概念外，也透過各種方法與工具協助讀者工作與生活後打造多元工作組合。

人生不是一條直線，人生反而更像是一條蜿蜒曲折的道路，所有追求美好的努力都會以不同的視野回報我們。知道自己想要什麼只是成功的一半，另一半是：如果當前的情況不符合期望，可以自由地選擇離開。改變、學習，幾歲都可以！只是要如何開始、怎麼開始？所有的第一步都是：停止抱怨！

我從這本書獲得什麼？

一、打造多元工作組合的四大支柱

  1. 身份不受市場的不可預測性影響
  2. 有選擇性的追求各種機會，無需受到傳統線性職涯的限制
  3. 多樣化的資源以降低個人和體制風險
  4. 能夠根據實際需求安排時間、平衡工作與生活的彈性

1. 身份

現代人習慣以工作頭銜或是職稱作為自己的身份代表，但是你不等同你的工作。應當要思考的是「當沒了名片，我還剩下什麼？」來更認識自己的身份。自我介紹的目標不是依序（一口氣）背誦三分鐘的簡歷，而是分享你能提供哪些解決方案。

2. 選擇性

在任何時候都有許多不同的道路可以選擇。即使有些路行不通，新的道路也會在每個關鍵時刻出現。無需被傳統的線性職涯框架限制，就在此刻也許就有新的職業出現，擁有追種機會的選擇性。

3. 多樣化

只從事一份工作可能是你這輩子最冒險的職涯選擇，用多樣化降低風險。科技不停發展，如果只一直從事同一門工作可能會遇到殺手級應用取代現在人力。

4. 彈性

讓你能夠重新調整承諾，重新平衡有薪工作、無薪工作、興趣、家庭責任和社區的權重，以適應新的情況。全職生涯可以轉型成為自由業，興趣愛好可以發展成爲小型事業。

二、建構自己的投資組合人生

當主要事物或需求轉變時，可以重新平衡或重建投資組合。

鍛鍊自己接受失敗的能力：習慣在小地方失敗，以免遭受重大的挫敗感

投資組合的基本前提是：有些事情可能不會成功，但成功的部分將會抵銷損失。

三、個人董事會

在自己身邊的朋友裡尋找協助做決策的董事會成員，通常包含五種角色：

1. 教練

能提出正確的問題，幫助你找到最適合的決定。

2. 談判者

掌握當前市場情報的人，幫助你爭取自身權益的關鍵夥伴。

3. 中介者

人與人聯繫的「超節點」，也就是常說的人脈。

4. 加油打氣者

分享生活中的高峰和低谷，提供情感支持的人。

5.說真話者

用你聽得進去的方式指出你的錯誤，並以你的最大利益為重，鞭策你做得更好。
需要是個真正了解你的人，如果你不確定對方是不是，那就不是。

四、兩種思維模式

1. 發散模式

收集訊息，從截然不同的觀點中求靈感，以便發現新的行為模式、解決問題的技巧、邊緣趨勢和外部力量的早期信號。這階段的重點不在於預設未來，而是保持開放的心態尋找潛在變化的線索、人們可能如何處理，以及誰可能受到這些事件的影響。

2. 聚焦模式

對發散搜集來的線索和跡象進行批判性分析，建構出應該會發生、合理會發生和有可能會發生的情景。這使我們能夠尋找數據、考慮時間軸，並提前規劃可能的應對策略。

閱讀後進行了什麼行動？

一、定義自己的成功

如果在某方面這麼出色，就要全力以赴！

在現實世界中不再重視或鼓勵均衡發展了，應該努力追求令人印象深刻的偏向發展，以此為基礎建立你的職業生涯。用自己的標準重新界定「成功」的意義，不要把自己的身份定義權交到別人手中。

二、衡量工作與生活

對企業來說最重要的是利潤，對個人來說最重要的是幸福

我們的幸福絕大部分取決於我們每天做的選擇，這些選擇是關於如何投資自己的時間、如何優先處理無窮盡的責任和看似不可能的權衡決策，而不是我們在自己領域中晉升到最高職位的能力。

三、使用文氏圖更了解自己：

文氏圖必然會隨著生活經歷不斷變化，適時地增加更多經歷、淘汰不適合的

圓圈包含以下元素：

1. 你目前正在從事哪些活動？

不只是職稱，寫下實際負責的內容。也可以思考其他生活的層面。

2. 你還參與過哪些活動？

過去參與過的活動，不一定是真正的「工作」。

3. 你對什麼極度著迷？

有空時都喜歡做什麼？關注的人？演算法會推薦什麼給你？活躍在什麼社群平台？

4. 人們對你有什麼特別的印象？

什麼情況下朋友會來尋求你的協助？不僅僅限於專業方面，並回想自己比同儕脫穎而出的地方。

5. 找出共同點並分類

  1. 將不喜歡的事情放在一疊
  2. 從喜歡的事情裡針對希望多做的事情加上星號
  3. 在兩堆中尋找模式和重疊之處，將看似不同生活層面整理成明確的主題

6. 整理所有主題並尋找交點，畫出最大的文氏圖

這本書適合什麼樣的人閱讀？

對職涯感到懷疑，或是處在職涯分岔點的人非常推薦閱讀這本書。更加了解自己並且從自身幸福的角度建構多元、有彈性的職涯組合，而不是傳統的線性框架。
